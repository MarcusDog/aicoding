# Thesis Blueprint

## Title

基于Web的大学生志愿者管理系统设计与开发

## Research Object

以上海立达学院高校志愿服务管理场景为背景，围绕学生端与管理员端双角色业务，构建一套覆盖活动发布、报名审核、签到签退、时长认定、公告通知与统计报表的 Web 志愿者管理系统。

## Research Goal

- 从高校志愿服务信息化的现实需求出发，梳理大学生志愿服务管理的核心业务流程。
- 结合项目源码，给出基于 Vue 3 与 Spring Boot 的 B/S 架构系统设计方案。
- 通过数据库设计、模块划分、关键代码实现与测试分析，验证系统的可行性、实用性与可维护性。
- 形成符合上海立达学院本科毕业论文规范的完整论文文档，做到结构完整、语言规范、图表齐全、参考文献真实可信。

## Technical Route

1. 阅读任务书、开题报告与学校写作规范，确定论文边界与格式约束。
2. 分析项目源码、数据库脚本和运行结果，提炼真实可写的系统事实。
3. 围绕需求分析、架构设计、数据库设计、功能实现、系统测试五个层面组织论文正文。
4. 采集系统截图、绘制架构图与 ER 图，补充图表、公式和核心代码片段。
5. 基于模板生成工作文档，完成正文装配、目录生成、样式修复与最终审校。

## Chapter Plan

1. 绪论
2. 相关技术与理论基础
3. 系统需求分析与数据库设计
4. 系统架构与功能模块设计
5. 系统实现与关键技术分析
6. 系统测试与结果分析
7. 结论与展望
