# 证据目录

这个目录只放真实证据，不放临时草稿。

## 建议子目录

- `evidence/source/`：下载证明、数据来源截图、API 文档截图。
- `evidence/raw/`：原始导出文件或样本。
- `evidence/processed/`：清洗统计、特征表、模型结果。
- `evidence/screenshots/`：系统截图。
- `evidence/questionnaire/`：问卷或访谈材料。
- `evidence/notes/`：标注规则、补充说明。

## 最低要求

- 每份证据都能说明来源。
- 论文中使用的图表要能回到原始文件。
- 版本更新后保留旧证据，不要覆盖掉原始记录。
