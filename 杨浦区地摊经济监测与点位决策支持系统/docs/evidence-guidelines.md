# 证据目录说明

## 目录职责

- `evidence/source/`：数据来源截图、下载证明、API 文档截图。
- `evidence/raw/`：原始样本文件或原始导出表。
- `evidence/processed/`：清洗统计、特征表截图、模型结果表。
- `evidence/screenshots/`：系统运行截图。
- `evidence/questionnaire/`：问卷原始回收结果。
- `evidence/notes/`：标注规则、访谈记录、补充说明。

## 文件命名

- `YYYYMMDD_source_xxx.png`
- `YYYYMMDD_cleaning_summary.xlsx`
- `YYYYMMDD_model_result.csv`
- `YYYYMMDD_screen_01.png`

## 归档要求

1. 先存原始证据，再整理成论文图表。
2. 截图必须来自真实运行界面。
3. 文档中引用的每一张图都要能回到目录里的原始文件。
