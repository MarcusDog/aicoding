# 测试计划

## 测试目标

验证三件事：

1. 数据链条真实可追溯。
2. 推荐结果能复算。
3. 系统能够本机稳定演示。

## 测试分类

| 类别 | 关注点 | 输出物 |
| --- | --- | --- |
| 数据测试 | 字段完整性、去重、缺失值、坐标有效性 | 清洗统计表 |
| 模型测试 | 规则评分、逻辑回归、Top3 稳定性 | 结果表、对照表 |
| 系统测试 | 启动、筛选、刷新、导出、异常输入 | 测试记录 |
| 文档测试 | 证据可追溯、图表来源清楚 | 证据目录 |

## 推荐测试用例

| 编号 | 测试项 | 输入 | 期望结果 |
| --- | --- | --- | --- |
| TC-01 | 原始数据检查 | 投诉 CSV | 关键字段存在 |
| TC-02 | 坐标有效性 | 点位 CSV | 经纬度可解析 |
| TC-03 | 清洗结果 | 原始投诉数据 | 去重和缺失统计正确 |
| TC-04 | 特征生成 | 清洗后数据 | 生成特征表 |
| TC-05 | 模型输出 | 特征表 | 生成 `predictions.csv` |
| TC-06 | 页面启动 | Streamlit | 页面可访问 |
| TC-07 | 导出功能 | 推荐结果 | 成功导出 CSV |
| TC-08 | 异常输入 | 缺失字段文件 | 给出明确提示 |

## 验收门槛

- 任何结论都能回到原始数据或中间结果。
- Top3 推荐可以复算。
- 页面在本机可以完整打开。
- 论文中的图表能在 `evidence/` 中找到原始来源。
