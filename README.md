# aicoding

项目汇总仓库。

当前项目：

- `shanghai-house-price-analysis-system/`：基于大数据的上海市二手房价数据分析系统，包含 Django 后端、Vue 3 前端和数据采集脚本。
- `杨浦区地摊经济监测与点位决策支持系统/`：面向街区治理管理者和一线巡查人员的地摊经济监测与点位决策支持原型，包含 Python 数据处理流水线、Streamlit 可视化系统和证据链材料。
- `基于WEB的家教信息发布和求职平台/`：基于 Spring Boot 3、Vue 3 和 Python 统计模块的家教信息发布与求职平台，包含后端、前端、数据库脚本和测试。
- `基于Spark的新闻热度监测系统/`：基于 Spark 的新闻热度监测系统，包含 Flask 后端、Vue 3 前端和新闻采集分析脚本。
